    How to get the file

    How do I download my Instagram data?
    Request the ZIP with your followers/following .json files:
    1. Tap the ☰ menu in the top right of Instagram
    2. Accounts Center → Your information and permissions
    3. Export your information → Create export
    4. Export to device → Customize information
    5. Check only Followers and Following
    6. Set Date Range to All time
    7. Format → JSON → Start export
    Instagram emails the ZIP when it's ready (minutes to ~48 hours). Upload it here as-is — no need to unzip.
    Do I need to give you my IG login?
    No. The only thing you provide is the ZIP file, which contains who you follow and who follows you. All reading and comparing happens locally in your browser — the data never touches a server.
    It says my file is missing the lists.
    Make sure you chose JSON (not HTML) and checked Followers and Following when exporting. The needed files live in connections/followers_and_following/ as followers_1.json (sometimes split into _2, _3…) and following.json. If you exported HTML, request a new JSON export.
