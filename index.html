Request your data from Instagram

This opens Instagram's official export page. Pick these exact options and it finishes in minutes, not days.

  1. Tap the button below to open Accounts Center → Export your information.
  2. Choose Create export → Export to device.
  3. Hit Customize information and check only Followers and Following.
  4. Set Date range to All time.
  5. Set Format to JSON — this matters. HTML won't work.
  6. Tap Start export. Instagram emails you a ZIP, usually within minutes.
